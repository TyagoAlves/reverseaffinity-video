# reverseaffinity Photo — Atalhos de Teclado (Keyboard Shortcuts)

Referência completa de atalhos de teclado, organizada por categoria.

---

## File Operations (Operações de Arquivo)

| Atalho | Ação | Categoria |
|--------|------|-----------|
| `Ctrl+N` | Novo documento (New) | File |
| `Ctrl+O` | Abrir imagem (Open) | File |
| `Ctrl+S` | Salvar (Save) | File |
| `Ctrl+Shift+S` | Salvar como (Save As) | File |
| `Ctrl+Q` | Fechar aplicação (Quit) | File |

## Edit (Editar)

| Atalho | Ação | Categoria |
|--------|------|-----------|
| `Ctrl+Z` | Desfazer (Undo) | Edit |
| `Ctrl+Shift+Z` | Refazer (Redo) | Edit |
| `Ctrl+C` | Copiar (Copy) | Edit |
| `Ctrl+V` | Colar (Paste) | Edit |

## View (Visualização)

| Atalho | Ação | Categoria |
|--------|------|-----------|
| `Ctrl++` | Zoom In | View |
| `Ctrl+-` | Zoom Out | View |
| `Ctrl+0` | Zoom para ajustar (Fit to Screen) | View |
| `Ctrl+1` | Zoom 100% | View |

## Tools (Ferramentas)

### Seleção

| Atalho | Ferramenta | Categoria |
|--------|------------|-----------|
| `V` | Move Tool (Mover) | Select |
| `M` | Rectangular / Elliptical Select | Select |
| `L` | Lasso Tool (Laço) | Select |
| `W` | Magic Wand (Varinha Mágica) | Select |

### Desenho

| Atalho | Ferramenta | Categoria |
|--------|------------|-----------|
| `B` | Brush Tool (Pincel) | Draw |
| `N` | Pencil Tool (Lápis) | Draw |
| `P` | Pen Tool (Caneta / Bezier) | Draw |
| `E` | Eraser Tool (Borracha) | Draw |
| `G` | Gradient Tool (Gradiente) | Draw |
| `U` | Shape Tool (Forma) | Draw |

### Texto

| Atalho | Ferramenta | Categoria |
|--------|------------|-----------|
| `T` | Text Tool (Texto) | Text |

### Retoque

| Atalho | Ferramenta | Categoria |
|--------|------------|-----------|
| `S` | Clone Stamp (Carimbo) | Retouch |
| `J` | Healing Brush (Pincel de Cura) | Retouch |

### Corte

| Atalho | Ferramenta | Categoria |
|--------|------------|-----------|
| `C` | Crop Tool (Corte) | Crop |

### Cor

| Atalho | Ferramenta | Categoria |
|--------|------------|-----------|
| `I` | Color Picker (Conta Gotas) | Color |
| `K` | Flood Fill (Balde de Tinta) | Color |

### Visualização

| Atalho | Ferramenta | Categoria |
|--------|------------|-----------|
| `H` | Hand Tool (Mão / Pan) | View |
| `Z` | Zoom Tool | View |

## Layer (Camadas)

| Atalho | Ação | Categoria |
|--------|------|-----------|
| `Ctrl+Shift+N` | Nova camada (New Layer) | Layer |

## Brush Size (Tamanho do Pincel)

| Atalho | Ação | Categoria |
|--------|------|-----------|
| `[` | Diminuir tamanho do pincel | Brush |
| `]` | Aumentar tamanho do pincel | Brush |

## Special Keys (Teclas Especiais)

| Tecla | Ação | Categoria |
|-------|------|-----------|
| `Enter` / `Return` | Finalizar path (Pen) / Aplicar corte (Crop) | Tool Action |
| `Esc` | Cancelar path (Pen) / Cancelar corte (Crop) | Tool Action |
| `Alt` + clique | Definir origem (Clone Stamp / Healing Brush) | Tool Modifier |
